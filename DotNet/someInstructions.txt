1. How to create a layout page ?
Views-->Shared-->Add--> New item-->Web--> MVC-->MVC 5 layout page(Razor)-->give proper name-->Add

2.To create other views which will be added as placeholders to the layout page
Views--> AnyControllerFolder-->Create view--> check the box "using layout page"--> 
on three dots i.e. browse button, select the layout view you want as your layout page--> Add

3. To add WebApi controller ?
Controllers folder--> add--> new item--> Web-->Web API-->Web API controller class --> give appropriate name and add.

4. how to craete Web API application ?
solution --> new project--> select ASP.NET web application(.NET framework)-->put name of project-->create--> 
select "Web API"--> UNCHECK the "configure for https" checkbox(since it needs license)-->create

5. how to craete MVC application ?
solution --> new project--> select ASP.NET web application(.NET framework)-->put name of project-->create--> 
select "Empty"--> CHECK MVC checkbox && UNCHECK the "configure for https" checkbox-->create

6. How to add enitity framework?
Wherever you want to add the framework, e.g. in Models folder of MVC application.
Models--> add new item--> Data-->ADO.NET entity data model-->put the model name below, same as that of the database name,for easier recognition in future-->
Add-->EF designer from database-->If not already connected to the correct database--> add new connection(steps for doing that we know already,its easy)--> verify if it's the desired connnection you want-->
check the connection string once if needed-->keep the checkbox for save connection settings in Web.config as "checked"-->next-->select the entity framework(choose 5.0 if wanna have stable one)-->
keeping all checkboxes as they are i.e. "checked", just select the tables you wanna connect to from the db connected-->Finish
